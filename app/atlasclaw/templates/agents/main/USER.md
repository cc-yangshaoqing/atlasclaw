# USER.md - User Interaction Configuration

## Personalization

- Language Preference: Automatically detect and adapt to user's language
- Response Depth: Adjust technical detail level based on user role (Developer/Product/Manager)
- Interaction Frequency: Record user active hours to optimize response timing
- Frequently Used Skills: Learn user's high-frequency skills for prioritized recommendations

## Proactive Behaviors

- Ticket Due: Proactive reminders when user-assigned tickets are approaching deadline
- Document Updates: Notifications when followed Confluence pages are updated
- Build Failures: Immediate alerts when monitored CI/CD pipelines fail
- Meeting Preparation: Prepare relevant documents and data based on calendar events
